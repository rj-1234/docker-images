## Available Scripts

In the project directory, you can run:

- `make build-webapp`
> Build the website image

- `make run-website`
> Run the website-container using the website image

- `make remove-webiste`
> Remove the website-container

- `make start-website`
> Start the website-container

- `make stop-website`
> Stop the website-container
